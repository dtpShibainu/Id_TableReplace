# TableReplace
Text Replaced to Table for InDesign

>External text sources (.txt, Excel, Word)<br>
>Text inside InDesign text frames<br>
>Tables

After copying the text from any of these sources, select the cells of the table where you want to flow the text.

>Using the script “Table Replace 列.jsx” will flow the text column-wise starting from the top-left cell.<br>
>Using the script “Table Replace 行.jsx” will flow the text row-wise starting from the top-left cell.

If there are merged cells, the behavior may become unstable.<br>
We recommend using simple tables.
